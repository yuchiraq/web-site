# Сайт ЧСУП «АВАЮССТРОЙ» (Брест и Брестская область)

Официальный сайт компании по строительным работам и инженерным услугам: автономная канализация (ТОПАС), водопонижение, дренаж, ливневая канализация, аренда техники.

## Что сделано в текущем обновлении

### 1) Улучшены ключевые страницы услуг под рекламу и конверсию
Обновлены страницы:
- `/services/topas`
- `/services/water_lowering`

Добавлено:
- сильный первый экран (оффер + выгоды + CTA);
- CTA для звонка, расчета, Telegram/Viber;
- блоки доверия, кейсы, этапы работ, FAQ;
- быстрые формы (префилл в модальное окно);
- география работ по Бресту и области.

### 2) Улучшено SEO-оформление на страницах услуг
На обновленных страницах реализовано:
- корректные `title`, `description`, `h1/h2`;
- `canonical`, `robots`;
- OpenGraph + Twitter card;
- JSON-LD разметка: `LocalBusiness`, `Service`, `FAQPage`, `BreadcrumbList`;
- хлебные крошки в контенте;
- более естественные продающие тексты без переспама.

### 3) Добавлен/усилен трекинг конверсий
В `static/js/script.js` добавлена отправка событий в `dataLayer` с fallback для `gtag` и `ym`:
- `phone_click`
- `form_submit`
- `order_click`
- `messenger_click`

Поддерживаются клики по:
- кнопкам CTA с `data-track`
- ссылкам `tel:`
- Telegram/Viber ссылкам
- отправке основной формы обратной связи

### 4) Проверка Go-файлов автогенерации SEO
Проверены файлы:
- `sitemap.go` — добавлено исключение технических шаблонов `templates/static/*` и `404.html` из sitemap.
- `generateRSS.go` — использует заголовки и description из HTML; рекомендуется продолжить унификацию SEO-мета по всем шаблонам.

---

## Запуск проекта

```bash
go mod tidy
go run .
```

Открыть: `http://localhost:8088`

---

## Архитектура (кратко)

- Backend: Go + Gin.
- Frontend: HTML templates + CSS + JS.
- Шаблоны: `templates/`.
- Статика: `static/`.
- Карта сайта: генерируется при старте (`sitemap.go`).
- RSS/Yandex Turbo: отдается через `/rss.xml` (`generateRSS.go`).

## Важные замечания для разработки

- В шаблонах использовать семантические секции (`main`, `section`, `article`, `nav`) и 1 H1 на страницу.
- Новые CTA-кнопки маркировать `data-track` для аналитики.
- Для локального SEO сохранять акцент на гео: Брест + города области (без переспама).
